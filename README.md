# optimal_photosynthetic_strategies

## Repository Description
This repository contains R code necessary for simulating optimal photosynthetic strategies
as presented in the manuscript *Optimal photosynthetic strategies* (Smith, unpublished).

## Summary of files and folders
The script [optimal_photosynthetic_strategies.R](optimal_photosynthetic_strategies.R) contains
code for simulating optimal photosynthetic strategies. This code sources code from
the repository called [optimal_vcmax_R](https://www.github.com/smithecophyslab/optimal_vcmax_R).

The folder [data](data) contains data used in the original analysis.
